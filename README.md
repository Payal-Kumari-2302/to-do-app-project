📝 To-Do App  
A simple and responsive To-Do application built using HTML, CSS, and JavaScript.  
It helps users add, complete, and delete daily tasks easily.

🚀 Features  
Add New Tasks  
Mark Tasks as Completed  
Delete Tasks  
Responsive Design  
User-Friendly Interface  

🛠️ Technologies Used  
HTML  
CSS  
JavaScript
